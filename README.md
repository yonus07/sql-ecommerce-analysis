# SQL Ecommerce Analysis (SQLite)

This project analyzes an ecommerce dataset using SQL in SQLite.  
It includes queries for revenue insights, customer behavior analysis, product performance, and review exploration.

## Project Structure
- `data/ecommerce.db` – SQLite database
- `queries/ecommerce_queries.sql` – SQL queries used for analysis
- `images/` – screenshots of SQL outputs
- `README.md` – project documentation

## Dataset
The dataset contains:
- Customer details  
- Orders and order status  
- Products and categories  
- Quantity and pricing  
- Ratings and reviews  
- Order dates  

Main table used: **ecommerce_dataset**

## SQL Topics Covered
- Data preview and summarization  
- Revenue calculations  
- Top customers and top products  
- Monthly revenue trends  
- Category performance  
- Repeat customers  
- Review and rating analysis  
- Views and indexing for optimization  

## How to Run
1. Open **DB Browser for SQLite**
2. Load `data/ecommerce.db`
3. Open `queries/ecommerce_queries.sql`
4. Run queries in the **Execute SQL** tab
5. Save outputs/screenshots in the `images/` folder

## Summary
This project demonstrates SQL-based data analysis using a real ecommerce dataset, highlighting insights into customer spending, product trends, and overall business performance.
